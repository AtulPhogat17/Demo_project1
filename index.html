Git terminologies...

1. git status

Check the status of files

2. git init

The git init command initializes a new Git repository in the current directory.

It sets up Git tracking for the project by creating a hidden .git/ folder.

You can see it with: ls -a

3. git add (filename) or git add .

Add a file to the staging area

4. git commit -m "Your commit message"

Commit changes with a message

5. git log

View commit history

6. git log --oneline

Show condensed commit history

7. git branch

List all branches

8. git branch (branch-name)

Create a new branch

9. git checkout (branch-name)

Switch to a branch

10. git switch (branch-name)

Switch branches (modern alternative)

11. git checkout -b (branch-name)

Create and switch to a new branch

12. git merge (branch-name)

Merge a branch into the current branch

13. git branch -d (branch-name)

Delete a branch locally
